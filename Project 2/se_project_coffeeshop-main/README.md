Coffee Shop Landing Page (BEM + Flexbox)

A responsive landing page for a coffee shop, built as part of the TripleTen Full-Stack program.
The project focuses on clean, maintainable HTML/CSS architecture using BEM methodology, a flat file structure, and modern layout techniques.

Features

Semantic HTML structure with clear sections (menu, about, footer)

Flexbox-based layouts for complex components (menu grid and cards)

Smooth hover effects and transitions

Simple CSS animation in the “About” section (pulsing effect)

Refactored CSS into a flat BEM structure using modular block stylesheets

Tech Stack

HTML5

CSS3 (BEM, Flexbox, transitions, animations)

Git & GitHub

Project Structure (BEM flat structure)

blocks/ — separate stylesheet per BEM block

pages/index.css — contains only @import statements

vendor/normalize.css — imported first for consistent base styling

images/ — project assets

What I Practiced

Applying BEM naming conventions and organizing styles by blocks, elements, and modifiers

Refactoring a single CSS file into modular, maintainable stylesheets

Building consistent layout systems with Flexbox

Debugging file paths and CSS imports using browser DevTools
